# 🚀 Code Mentor AI (DSA Instructor)

An AI-powered web application that acts as a **Data Structures & Algorithms mentor**, helping users understand concepts in a simple and structured way.

---

## 🌐 Live Demo

https://code-mentor-smy0.onrender.com/

---
## 🔥 Features

* 🧠 **DSA Mentor Mode**

  * Answers only Data Structures & Algorithms questions
  * Provides clear, beginner-friendly explanations

* 📜 **History Tracking**

  * Stores previous questions and answers
  * Click to revisit past responses

* 📚 **Tutorials Section**

  * Predefined DSA topics (Array, Linked List, Stack, Queue)
  * One-click learning experience

* ⚙️ **Settings**

  * Clear chat history

* 🎨 **Modern UI**

  * Dark theme
  * Clean and user-friendly interface

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Flask (Python)
* **AI Integration:** Google Gemini API (`google-genai`)
* **Deployment:** Render

## ⚙️ Installation (Local Setup)

1. Clone the repository:

```
git clone https://github.com/your-username/code-mentor-ai.git
cd code-mentor-ai
```

2. Create virtual environment:

```
python -m venv venv
venv\Scripts\activate   # Windows
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Add API key:

```
set GOOGLE_API_KEY=your_api_key   # Windows
```

5. Run the app:

```
python app.py
```

6. Open in browser:

```
http://127.0.0.1:5000
```
